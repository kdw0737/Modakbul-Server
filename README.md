# [2024] Modakbul 🔥

# 📣 소개
스터디를 구할 때마다 한 명씩 팀원을 구하고, <br>
장소와 시간을 설정하는 데 불편함을 느껴본 적이 있으신가요? <br>
Modakbul은 이런 불편함을 해소하기 위해 <br>
지정된 장소에 스터디 모집글을 올리고 <br>
인원을 신청받아 더욱 편리하게 모집 과정을 진행할 수 있도록 하였습니다. 😊
