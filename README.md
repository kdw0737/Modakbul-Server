---

## 📝 소개
시험공부, 스터디 등 함께 공부할 사람을 찾을 수 있는 Modakbul 입니다!  
만남 장소와 카테고리를 선정하고 채팅을 통해 자유롭게 일정을 조정할 수 있습니다!
